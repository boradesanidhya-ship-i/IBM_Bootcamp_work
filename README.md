## Current Implementation

5G OptiMind is a web application developed as an AI-Based 5G Network Optimization Advisor.

The project currently uses the following IBM technologies:

* IBM Cloud
* IBM watsonx Orchestrate
* IBM Langflow
* IBM AI Models

The website also includes an integrated AI chatbot feature that allows users to interact with the AI system.

The application is designed around the concept of using AI to support 5G network optimization and analysis.
